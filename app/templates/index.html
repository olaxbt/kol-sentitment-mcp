Overview

KOL Sentiment MCP provides a standardized Model Context Protocol interface for AI assistants to analyze Key Opinion Leaders (KOLs) on social media platforms. It leverages the Masa AI API for powerful semantic search and data retrieval.

Features

  • Search KOL content across platforms
  • Analyze sentiment of KOL posts
  • Extract trending topics from KOL content
  • Generate comprehensive KOL sentiment analysis
  • Identify trends across multiple KOLs
  • Both live and historical data analysis

API Endpoints

GET /health - Health check endpoint
GET /api/mcp/ping - MCP API health check
POST /api/mcp/query - Main MCP query endpoint

Supported Actions

  • kol.search - Search for KOL content
  • kol.sentiment - Analyze KOL content sentiment
  • kol.topics - Extract topics from KOL content
  • kol.sentiment_analysis - Get comprehensive KOL sentiment analysis
  • kol.trends - Analyze trends across multiple KOLs
